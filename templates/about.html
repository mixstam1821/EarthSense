EarthSense is a powerful interactive visualization platform that represents the results of my PhD. It allows users to explore the data and insights from my study. With a range of features and tools, EarthSense empowers users to make informed decisions and gain a deeper understanding of the Global Dimming and Brightening, the causes of this phenomenon and the potential relationship with other climate variables.

  • General Climate Explorer
  • Dynamic updates
  • Rich interactions
  • Click or select region to get the corresponding data
  • Display large amounts of climatic data fast and in a user-friendly way
  • Comfortable viewing and Introduction

  • Bokeh - For creating interactive visualizations
  • FastAPI - Modern, fast web framework for building APIs
  • Python - Core programming language
  • HTML/CSS - Frontend styling with modern design principles
  • Jinja2 - Template engine for Python

Built with performance in mind, EarthSense leverages Bokeh's efficient data handling and FastAPI's high-speed backend to provide a smooth user experience even with large datasets.
